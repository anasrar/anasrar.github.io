## Notes

Only work on equirectangular[^projection] video.

```bash
git clone https://github.com/google/spatial-media spatial-media
cd spatial-media
python spatialmedia --inject <input file> <output file>
```


[^projection]: https://en.wikipedia.org/wiki/360_video_projection
