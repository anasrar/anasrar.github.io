## Information

### Stack

#### Mantine

[Mantine](//github.com/mantinedev/mantine/) is a beautiful [React](//github.com/facebook/react) components library.

#### Vite

[Vite](//github.com/vitejs/vite) is fast code bundler with hot reload, make it easy to develop front-end stuff.
