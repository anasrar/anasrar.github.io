Blender Unreal Engine 4 Workspace - Blender addon

Download Documentation Repository YouTube Playlist

Blender addon for export assets to Unreal Engine 4 with just clicking button.

Information

Feature

Static Mesh

Export for static mesh.

  • Export to FBX and Unreal Engine
  • Custom collision from vertices
  • Custom collision from mesh v1.2
  • Custom lightmaps
  • Export profile v1.2 DEPRECATED v2.0
  • Socket system v1.3
  • Level of detail v1.4
  • Import static mesh from Unreal Engine v1.4
  • Generate level of detail v2.0

Skeletal Mesh

Export for skeletal mesh.

  • Export to FBX and Unreal Engine
  • Modular characte
  • Skeleton preset (Epic skeleton) DEPRECATED v2.0
  • Add twist bone for skeleton preset v1.2 DEPRECATED v2.0
  • Generate rig for skeleton preset v1.2 DEPRECATED v2.0
  • Export profile v1.2 DEPRECATED v2.0
  • Socket system v1.3
  • Skeletal mesh part manager v1.4
  • Import skeletal mesh from Unreal Engine v1.4
  • Copy/Paste socket Unreal Engine v2.0

Animation

Export for animation.

  • Export to FBX and Unreal Engine v1.2
  • Export profile v1.2 DEPRECATED v2.0
  • Import animation from Unreal Engine v1.4

Retarget Animation

Retarget animation to another skeleton Experimental v1.3 Production Ready v1.4 DEPRECATED v2.0

Groom Hair [Experimental]

Export hair particle from Blender and import as groom hair in Unreal Engine (not Support direct export To Unreal) v2.0

Copy Transform To Unreal Engine Map

Copy transform selected object to Unreal Engine map v2.0
